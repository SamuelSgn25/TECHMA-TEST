## 🎙️ AUDIO ENGINEERING

### Microphone Check:
- Test at -6dB to -3dB (good levels)
- No clipping (don't max out volume)
- Minimal background noise

### Audio Enhancement:
- Compression (subtle, -4db ratio)
- EQ (slight high-pass filter above 80Hz)
- Noise gate (to remove background hum)

### Backup Audio:
- Record local audio file
- Upload audio + video separately
- Sync in post-production if needed

## 🎬 FINAL EDITING CHECKLIST

After recording, before uploading:

- [ ] Trim silences longer than 3 seconds
- [ ] Normalize audio levels
- [ ] Add background music (royalty-free)
- [ ] Add captions/subtitles
- [ ] Add video intro/outro (5-10 sec each)
- [ ] Add chapter markers at major sections
- [ ] Color correct if needed
- [ ] Add sound effects for transitions (optional)
- [ ] Export at 1080p 60fps minimum
- [ ] Test playback on multiple devices

## 💾 FILE FORMAT GUIDELINES

**Export Settings:**
- Codec: H.264
- Resolution: 1920x1080
- Frame Rate: 30fps (or 60fps for smooth motion)
- Bitrate: 8-12 Mbps
- Audio: AAC, 128kbps, 48kHz
- File Format: MP4

**File Size:** 300-600MB for 20-minute video
(Adjust bitrate if different)

## 🎯 VISUAL HIERARCHY

**For Slides:**
1. **Title** (Largest, bold)
2. **Key point** (Medium, accent color)
3. **Supporting details** (Smaller, lighter)
4. **Footer info** (Smallest, gray)

**Font Sizes:**
- Title: 48-60pt
- Main text: 32-40pt
- Details: 24-28pt
- Footer: 14-18pt
